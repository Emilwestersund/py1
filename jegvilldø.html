opg 1

Input og output: Python bruker input() for tekstinput og print() for output, mens JavaScript bruker prompt() for input og alert() for output. Syntaks: Python bruker elif i betingelser, mens JavaScript bruker switch eller else if. JavaScript har også semikolon som avslutter setninger, noe som ikke er nødvendig i Python."

opg 2

Variabler: Python bruker dynamisk typede variabler, mens JavaScript bruker nøkkelordene const eller let for variabeldeklarasjon. Operasjoner: Begge språkene bruker * for multiplikasjon, men JavaScript krever semikolon etter hver setning.
